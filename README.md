# Data Mining and Image Classifier 

This repository demonstrates how to minining data from web using 'Selenium' + Data and features creation. Then, training image classifier by pytorch transfer learning.


### Recommended:
* python=3.6
* selenium=3.141.0
* torch=1.5.1
* torchvision=0.5.0
* sklearn=0.20.1
* PIL=7.2.0
